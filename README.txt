[Install Dependencies]:

                     1. PYTHON INSTALLATION:-[version>=3.7]
                                        step1: Goto the following link: https://www.python.org/downloads/     
                                        step2: Download and install the python windows setup.
                     
		     2. FFMPEG INSTALLATION:-
		                        step1: Click the following link: https://www.geeksforgeeks.org/how-to-install-ffmpeg-on-windows/                                       to download the zip file of the latest version. 
		                        step2: Unzip this file by using any file archiver such as Winrar or 7z.
		                        step3: Rename the extracted folder to ffmpeg and move it into the root of C: drive
		                        step4: Now, run cmd as an administrator and set the environment path variable for ffmpeg by running the       
		                              following command: "setx /m PATH "C:\ffmpeg\bin;%PATH%""
		                        step5: Restart your computer
		                        Step6: verify the installation by running : "ffmpeg -version"
		     3. OTHER INSTALLATIONS:-
		                          step1- pip install -r requirement.txt
